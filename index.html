All EVM Formulas
Schedule Variance
SV
SV = EV − PV
Positive = ahead of schedule. Negative = behind schedule. Measured in dollars — not days.
SV > 0 → Ahead ✅ SV < 0 → Behind ❌
Cost Variance
CV
CV = EV − AC
Positive = under budget. Negative = over budget. Compares value earned to money spent.
CV > 0 → Under ✅ CV < 0 → Over ❌
Schedule Performance Index
SPI
SPI = EV ÷ PV
How much value delivered per $1 of planned work. Above 1.0 = efficient. Below 1.0 = behind.
SPI > 1.0 → Ahead ✅ SPI < 1.0 → Behind ⚠️
Cost Performance Index
CPI
CPI = EV ÷ AC
Value earned per $1 spent. The most critical EVM index. Above 1.0 = under budget per unit.
CPI > 1.0 → Efficient ✅ CPI < 1.0 → Over budget ❌
Estimate at Completion
EAC
EAC = BAC ÷ CPI
Projected total cost at completion assuming current CPI continues. The key forecast number.
EAC ≤ BAC → On budget ✅ EAC > BAC → Overrun ❌
Estimate to Complete
ETC
ETC = EAC − AC
How much more money is needed from today to finish. EAC is total. ETC is the remaining portion.
ETC = remaining spend
Variance at Completion
VAC
VAC = BAC − EAC
Projected surplus or overrun at end of project. Positive = surplus. Negative = overrun projected.
VAC > 0 → Surplus ✅ VAC < 0 → Overrun ❌
To-Complete Performance Index
TCPI
TCPI = (BAC−EV) ÷ (BAC−AC)
CPI needed on remaining work to stay within BAC. If TCPI > current CPI — must work more efficiently.
TCPI ≤ CPI → Achievable ✅ TCPI > CPI → Harder ⚠️

Classic Exam Traps
⚠️
Comparing AC to PV directly
Budget performance uses CV = EV − AC. Never compare AC to PV. That comparison has no meaning in EVM.
⚠️
Thinking SV is measured in days
SV is in dollars — it measures planned value vs earned value in money terms. It is NOT calendar days.
⚠️
Confusing EAC with ETC
EAC = total projected cost at completion. ETC = remaining cost from today. ETC = EAC − AC.
⚠️
TCPI formula — wrong denominator
TCPI = (BAC−EV) ÷ (BAC−AC). Numerator = remaining work value. Denominator = remaining budget.
